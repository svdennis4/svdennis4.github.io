Help Documentation

Education World html template made by pure Bootstrap framework. You can easily modify this template with bit knowledge of coding. Have a look into brief modifications you should know before editing this html template

Core Steps

After successful download of this HTML Template, Extract it, open folder and copy 'Education World-HTML-Template' (this folder is actual template rename it and use

Files Structure
  • Html Files- (like index.html, about-us.html etc)
  • Assets- (This folder contains all frameworks and jQuery Plugins like Bootstrap, Font-awesome...Etc)
  • css- (This folder contains main style sheet i.e style.css)
  • images- (This folder contains all images of template)
  • js- (This folder contains main javascript file i.e custom.js)
Steps to edit this template
  • Create folder with name html
  • Download this template files and paste inside the html folder
  • We are not providing the images only image sizes
  • Please check and upload your images with mention size
  • Replace your images in style.css file wherever you want to call images to replace this code ("../images/about-img.jpg or .png"), also place all your images inside images folder
  • In header section in slider please change the images and text and use
  • Login to your server and upload the files inside html folder to public_html for Linux server
